# 信号合成上位机软件套装使用手册

## 1. 概述

信号合成上位机软件套装是一个集成的信号处理和通信平台，整合了以下三个核心模块：

1. **手绘波形生成器** - 通过可视化界面设计任意波形
2. **调制信号生成器** - 生成各种调制信号（ASK/BPSK/AM/FM）
3. **UDP通信助手** - 高速网络传输工具

## 2. 快速启动

### 2.1 启动服务器
双击 `启动服务器.bat` 文件，看到以下信息表示启动成功：
```
Starting UDP WebSocket Proxy Server...
OK: WebSocket server started on port 9000
```

### 2.2 打开主界面
在浏览器中打开 `index.html` 文件。

## 3. 功能模块说明

### 3.1 🎨 手绘波形生成器
- **功能**：在画布上用鼠标自由绘制波形，支持实时预览。
- **参数设置**：
    - 采样率
    - 采样点数
    - 周期数
- **导出**：点击"导出Bin文件"按钮，将生成的波形数据保存为 `.bin` 文件。

### 3.2 📡 调制信号生成器
- **功能**：生成标准调制信号。
- **支持类型**：ASK、BPSK、AM、FM。
- **参数设置**：
    - 载波频率
    - 采样率
    - 采样点数
    - 调制参数（幅度、相位、频偏等）
- **导出**：点击"导出Bin文件"按钮，将生成的信号数据保存为 `.bin` 文件。

### 3.3 🌐 UDP通信助手
- **功能**：通过UDP协议发送数据文件到FPGA开发板或其他设备。
- **连接设置**：
    - **本地IP地址**：本机IP地址。
    - **本地端口**：本机监听端口。
    - **远程主机**：目标设备（如FPGA）的IP地址。
    - **远程端口**：目标设备的接收端口。
- **操作流程**：
    1. 设置好IP和端口参数。
    2. 点击 **"连接"** 按钮。如果连接成功，状态栏会显示绿色"已连接"。
    3. 在"发送数据"区域，点击 **"选择文件"**，选择之前生成的 `.bin` 文件。
    4. 点击 **"发送文件"** 按钮，即可将数据传输至目标设备。

## 4. 常见问题与故障排除

**问题：无法连接UDP**
- 检查 `启动服务器.bat` 是否已运行且未报错。
- 检查端口9000是否被占用。
- 确认填写的IP地址和端口号正确。
- 检查防火墙设置，允许Node.js通过防火墙。

**问题：发送失败**
- 确保UDP连接状态为"已连接"。
- 确保选择了正确的文件格式（推荐使用.bin文件进行FPGA通信）。

**问题：FPGA未收到数据**
- 确认FPGA开发板的IP地址和端口与软件中设置的"远程主机"和"远程端口"一致。
- 检查网线连接是否正常。
